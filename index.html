# Name Description
1 Синхронный Браузер загружает страницу постепенно. Важно! Если скрипт-внешний, то пока браузер не выполнит его, он не покажет остальную часть страницы
2 async Скрипт выполняется полностью асинхронно с остальной веб-страницей. То есть, при обнаружении script async src="..." браузер не останавливает обработку страницы, а спокойно работает дальше. Когда скрипт будет загружен – он выполнится.
3 defer Скрипт также выполняется асинхронно, не заставляет ждать страницу, но есть два отличия от async. 1 -браузер гарантирует, что относительный порядок скриптов с defer будет сохранён. 2 - скрипт с defer сработает, когда весь HTML-документ будет обработан браузером.
